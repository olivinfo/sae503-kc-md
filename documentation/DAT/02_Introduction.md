# 1. Introduction

## 1.1 Contexte du projet

Ce projet s'inscrit dans le cadre de la SAE 5.03 (Situation d'Apprentissage et d'Évaluation) du programme informatique. L'objectif principal est de concevoir, développer et déployer une application web moderne basée sur une architecture de microservices.

L'application "Citations Haddock" est une plateforme permettant de gérer des citations de personnages célèbres, avec des fonctionnalités de recherche, d'ajout et de gestion des utilisateurs.

## 1.2 Objectifs

Les objectifs principaux du projet sont :

1. **Architecture moderne** : Concevoir une architecture basée sur des microservices
2. **Orchestration de conteneurs** : Utiliser Kubernetes pour l'orchestration et la gestion des conteneurs
3. **Sécurité** : Implémenter des mesures de sécurité robustes
4. **Scalabilité** : Créer une solution évolutive et maintenable
5. **Documentation** : Fournir une documentation complète de l'architecture et des choix techniques

## 1.3 Périmètre

Le projet couvre les aspects suivants :

- **Développement** : Refactorisation de l'application existante en microservices
- **Infrastructure** : Configuration de Kubernetes et des outils associés
- **Sécurité** : Mise en place de mesures de sécurité conformes aux exigences
- **Déploiement** : Automatisation du déploiement et de la configuration
- **Documentation** : Rédaction du dossier d'architecture technique

**Hors périmètre** :
- Développement de nouvelles fonctionnalités métiers
- Migration vers d'autres technologies de base de données
- Intégration avec des systèmes externes

## 1.4 Public cible

Ce document s'adresse à :

- **Équipe de développement** : Pour comprendre l'architecture et les choix techniques
- **Équipe d'exploitation** : Pour le déploiement et la maintenance
- **Responsables techniques** : Pour l'évaluation des choix architecturaux
- **Étudiants et enseignants** : Pour la compréhension du projet dans un cadre pédagogique
